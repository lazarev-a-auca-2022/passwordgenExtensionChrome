# Генератор паролей — расширение для Chrome (Go)

Расширение для Chrome, генерирующее надёжные пароли. По функциональности похоже на LastPass Password Generator. Основная логика написана на **Go** (компилируется в WebAssembly), UI — минималистичный попап.

---

## Основные функции

| Функция | Описание |
|---|---|
| Длина пароля | Слайдер, диапазон 4–64 символа |
| Заглавные буквы | Переключатель (A–Z) |
| Строчные буквы | Переключатель (a–z) |
| Цифры | Переключатель (0–9) |
| Символы | Переключатель (!@#$%^&* и т.д.) |
| Исключить неоднозначные | Переключатель (убирает `0`, `O`, `l`, `1`, `I`) |
| Индикатор надёжности | Визуальная полоса: Слабый / Средний / Сильный / Очень сильный |
| Копирование в один клик | Копирует пароль в буфер обмена |
| Перегенерировать | Генерирует новый пароль с текущими настройками |

## Сборка Go → WASM

Go экспортирует функции в JavaScript через биндинги `syscall/js`:

```go
// cmd/wasm/main.go
js.Global().Set("generatePassword", js.FuncOf(generatePassword))
js.Global().Set("scoreStrength",    js.FuncOf(scoreStrength))
```

**Команда сборки:**
```bash
GOOS=js GOARCH=wasm go build -o extension/main.wasm ./cmd/wasm/
cp "$(go env GOROOT)/misc/wasm/wasm_exec.js" extension/
```

## Ответственность файлов

### `internal/generator/generator.go`
- `GeneratePassword(length int, opts Options) string` — чистый Go, `crypto/rand`
- `ScoreStrength(password string) int` — оценка на основе энтропии (0–4)

### `cmd/wasm/main.go`
- Регистрирует Go-функции в глобальной области JS
- Держит WASM-модуль живым через `select {}`

### `extension/popup.js`
- Инициализирует WASM-модуль при открытии попапа
- Читает состояние UI (переключатели, слайдер) → вызывает `window.generatePassword()`
- Отрисовывает результат и полосу надёжности
- Обрабатывает копирование через `navigator.clipboard.writeText()`

### `extension/manifest.json`
- Manifest v3
- `action.default_popup: "popup.html"`
- Разрешения: `clipboardWrite`
